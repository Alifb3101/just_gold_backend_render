# How to Upload Product Images

---

## UPLOAD STEPS

### Step 1: Select Image File
- Click "Choose Image"
- Pick your product image (JPG, PNG, or WebP)
- Max size: 5MB

### Step 2: Click Upload
- Click the "Upload" button
- Wait for confirmation

### Step 3: Done ✓

That's it. The system handles everything else automatically.

---

## THAT'S ALL YOU NEED TO DO

✅ Select image file  
✅ Click upload  
✅ Done

The backend processes and stores it. You don't need to think about anything else.
